Typing Speed Tester

A simple Python program that calculates typing speed (WPM) and accuracy.

## Features

* Measures time taken to type a sentence
* Calculates Words Per Minute (WPM)
* Calculates accuracy percentage

## How to Run

1. Run the Python file
2. Type the given sentence
3. View your speed and accuracy

## Formula

* WPM = (Words / Time in seconds) × 60
* Accuracy = (Correct characters / Total characters) × 100

## Conclusion

This project demonstrates basic Python concepts like functions, time calculation, and user input.
